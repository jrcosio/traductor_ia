# Samples

Este directorio contiene audios de referencia para benchmarks y pruebas manuales.

Archivos actuales:

- `base_silence.wav`: muestra corta de silencio a `16 kHz`, mono, usada por el benchmark base del Sprint 0.
- `asr/en_corto.wav`: muestra corta en inglés para benchmark del ASR.
- `asr/es_corto.wav`: muestra corta en español para benchmark del ASR.
- `asr/es_medio.wav`: muestra media en español para benchmark del ASR.
